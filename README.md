# Multi-Tool AI Application

This is a Python-based AI application with a dark-mode interface. It features:
* **Text Summarizer:** Condenses long paragraphs into key points.
* **Idea Generator:** Brainstorms concepts based on user prompts.
* **Simple Chatbot:** An interactive assistant powered by Google's Gemini AI.

Built using Gradio and the `gemini-1.5-flash` model.
